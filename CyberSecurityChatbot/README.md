# Cybersecurity Awareness Chatbot

## Description
A simple C# console chatbot that teaches users about cybersecurity.

## Features
- Voice greeting (add your own welcome.wav file)
- ASCII logo
- User interaction
- Cybersecurity responses
- Input validation
- Typing effect
- GitHub Actions CI

## How to Run
1. Open in Visual Studio
2. Add welcome.wav file
3. Run the program
